# Currency Exchange Rate App

แอปแปลงอัตราแลกเปลี่ยน กำหนดมูลค่าเงินต้นทาง เห็นมูลค่าปลายทาง และแสดงมูลค่าแลกกลับ

## ฟีเจอร์

- **กำหนดมูลค่าเงินต้นทาง และเห็นมูลค่าปลายทาง** – ป้อนจำนวนเงิน เลือกสกุลเงิน From/To แสดงผลทันที
- **แสดงมูลค่าแลกมา / แลกกลับ** – แสดงทั้งการแปลงไป (แลกมา) และการแปลงกลับ (แลกกลับ)

## เทคโนโลยี

- HTML5, CSS3, Vanilla JavaScript
- [Frankfurter API](https://frankfurter.dev/) – ฟรี ไม่ต้องใช้ API key

## วิธีรัน

1. เปิดโฟลเดอร์ในเบราว์เซอร์ (หรือใช้ Live Server / http server)
2. เปิด `index.html` ในเบราว์เซอร์

ตัวอย่าง:

```bash
npx serve .
# หรือ
python -m http.server 8080
```

จากนั้นเปิด `http://localhost:8080` (หรือพอร์ตที่ใช้)

## การใช้งาน

1. กรอก **จำนวนเงินต้นทาง** ในช่องซ้าย
2. เลือก **สกุลเงินต้นทาง** (From) และ **สกุลเงินปลายทาง** (To)
3. ดู **เงินปลายทาง** ที่คำนวณให้อัตโนมัติ
4. ดูส่วน **แลกมา** และ **แลกกลับ** สำหรับสรุปการแปลง
5. กดปุ่มลูกศรเพื่อ **สลับสกุลเงิน** From/To
