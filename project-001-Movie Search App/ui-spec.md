

# Movie Search App — UI Spec

อ้างอิงจาก `ref.jpg` — ธีมมืด สไตล์ streaming แบบ minimalist

---

## 1. โครงร่างรวม (Layout)

- **Container:** กล่องหลักพื้นหลังมืด มุมโค้ง (rounded corners) อาจมีพื้นหลังเบลอ (blur) จากภาพ featured
- **สองส่วนหลัก:**
  1. **Hero / Featured** — ด้านบน เน้นหนึ่งเรื่อง (ภาพเต็ม + ชื่อ + ปุ่ม)
  2. **My List** — ด้านล่าง แถวโปสเตอร์เลื่อนแนวนอนได้

---

## 2. แถบนำทาง (Top Navigation)

| ตำแหน่ง | รายการ | รายละเอียด |
|--------|--------|------------|
| **ซ้าย** | ลิงก์นำทาง | DASHBOARD · MOVIES · SERIES · KIDS |
| **ขวา** | ไอคอน | แว่นขยาย (Search) · วงกลมโปรไฟล์ (Avatar) |

- **ตัวอักษร:** sans-serif, สีขาว
- **สถานะ active:** ข้อความหนาขึ้นหรือมีไฮไลต์ (เช่น พื้นหลังเข้มขึ้น) — ตัวอย่าง ref: "MOVIES" เป็น active
- **ไอคอน:** สีขาว, ปุ่มวงกลม, มี hover state

---

## 3. โซน Featured (Hero)

- **พื้นหลัง:** ภาพเต็มจากเรื่องที่ featured (เช่น Raya ในป่า/หมอก) เต็มพื้นที่ section
- **เนื้อหาซ้อนบนภาพ (overlay):**
  - **หัวเรื่องหลัก:** ตัวอักษรใหญ่ หนา สีขาว — ตัวอย่าง "RAYA AND THE LAST DRAGON"
  - **บรรทัดรอง:** ข้อความเล็กกว่า สีขาว — ตัวอย่าง "A Disney Original Film" ตามด้วย "98% Match 2021"
    - "98% Match" ใช้สีต่างออกไป (เช่น เขียวอ่อนหรือเหลือง) เพื่อเน้น
  - **ปุ่ม:**
    - **PLAY:** รูปไข่/rounded rectangle พื้นหลังม่วง ตัวอักษรขาว มีเอฟเฟกต์เงาหรือ gradient เล็กน้อย
    - **Add to list:** วงกลม ขอบขาว มีเครื่องหมาย + ขาว อยู่ข้าง PLAY

---

## 4. โซน My List

- **หัวข้อ:** ข้อความ "MY LIST" สีขาว หนา ชิดซ้าย
- **แถวโปสเตอร์:**
  - เลื่อนแนวนอนได้ (horizontal scroll)
  - โปสเตอร์แนวตั้ง อัตราส่วนแนวตั้ง (portrait) ขนาดสม่ำเสมอ
  - แต่ละการ์ดมีขอบหรือ glow สีขาวอ่อนให้แยกจากพื้นหลัง
  - ตัวอย่างใน ref: JOHN WICK, BABY DRIVER, BLADE RUNNER, BOHEMIAN RHAPSODY, BRAVE, CASABLANCA, CRAZY STUPID LOVE, DEADPOOL (ซีกขวาเห็นไม่เต็ม = บอกว่า scroll ได้)

---

## 5. สไตล์ภาพรวม (Visual Style)

| หัวข้อ | Spec |
|--------|------|
| **พื้นหลัง** | โทนมืด (ดำ/เทาเข้ม) |
| **ตัวอักษรหลัก** | สีขาว อ่านง่าย |
| **Accent** | ม่วง (ปุ่ม PLAY, จุดเน้น interactive) |
| **ตัวอักษร** | Sans-serif, หลายขนาด/น้ำหนักสร้าง hierarchy |
| **ความลึก** | เงาเล็กน้อยที่ container และการ์ดโปสเตอร์ |
| **ความสัมพันธ์กับ ref** | กล่องหลักอาจทับบนพื้นหลังเบลอจากภาพ featured |

---

## 6. สถานะและการโต้ตอบ (States)

- **Navigation:** มีสถานะ active (เช่น MOVIES ไฮไลต์) และ hover
- **ปุ่ม PLAY / Add:** มี hover (และถ้าต้องการ focus สำหรับ accessibility)
- **โปสเตอร์ใน My List:** มี hover (เช่น ขอบ/glow ชัดขึ้น หรือ scale เล็กน้อย) และคลิกได้

---

## 7. สรุป Component

| Component | หน้าที่ |
|-----------|--------|
| Nav bar | แท็บ DASHBOARD / MOVIES / SERIES / KIDS + Search + Profile |
| Hero | ภาพ featured + ชื่อเรื่อง + meta (98% Match ปี) + PLAY + Add |
| My List row | หัวข้อ "MY LIST" + แถวโปสเตอร์เลื่อนแนวนอน |
| Poster card | รูปโปสเตอร์ + (ถ้ามี) ชื่อ/ข้อมูลสั้น ขอบหรือ glow |

---

## 8. หมายเหตุสำหรับพัฒนา

- **Search:** จาก ref ไม่เห็นช่องค้นหา — ออกแบบเป็น overlay/panel เมื่อกดไอคอนแว่นขยาย
- **Modal รายละเอียด:** ไม่อยู่ใน ref — ใช้สไตล์เดียวกัน (พื้นหลังมืด, ตัวอักษรขาว, accent ม่วง)
- **Responsive:** ลดขนาดตัวอักษร/ปุ่มและจำนวนโปสเตอร์ที่เห็นในหนึ่งแถวบนมือถือ, แถว My List ยังเลื่อนแนวนอนได้
